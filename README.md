# WeatherAPI

A tool that grabs forecast data from the https://www.weatherapi.com/ API and stores it locally as a time series in a .csv file or database.

### Required data: 
##### Wind speed (Kmh)
##### Temperature (Celsius)
##### Cloud coverage (percent)
### At these locations:
##### Cologne
##### Munich 
##### Berlin
